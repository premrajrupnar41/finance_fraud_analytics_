# 💳 Credit Card Fraud Detection & Financial Risk Analytics

## Folder Structure
```
finance_fraud_analytics/
├── data/                  # Place dataset here
├── sql/schema.sql         # Schema + risk queries
├── notebooks/analysis.py  # Full ML pipeline
├── dashboard/powerbi_guide.md
├── reports/insights.md
└── README.md
```

## Dataset
Download: https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
- File: `creditcard.csv` → place in `data/` folder

## Setup
```bash
pip install -r requirements.txt
python notebooks/analysis.py
```

## Tools Used
Python · Pandas · NumPy · Scikit-learn · Imbalanced-learn · Matplotlib · SQL · Power BI
